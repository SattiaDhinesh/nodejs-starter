# Express starter with Mongoose and JsonWebToken.

- Simple starter for authentification with JWT and passportJS.  
- Jest for unit test .

## Installation

 Use the package manager [npm](https://nodejs.org/en/) to install and run the project.

```bash
npm install
npm start
```
Run test

```bash
npm run test
```
## Technologies
-  NodeJS (ExpressJS)
- Mongodb (Mongoose)
- PassportJS
- JsonWebToken
- Jest
